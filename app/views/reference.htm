Create a reference report

Replace all dynamic values, which change independently of the scanning result (e.g. target URL, timestamps, etc...), by placeholders. Placeholders will be ignored during the risk assessment evaluation, s.t. they don't cause any change of the risk-factor (i.e. they are going to be "ignored"). To indicate such a placeholder, use % to surround the dynamic value.
Example: I_WILL_CHANGE becomes %I_WILL_CHANGE%.
